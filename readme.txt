== Description ==

This plugin protected a page by recaptcha and it show the page after captcha is solved
in the wp-admin it need to enter 
1. google recaptcha version 2 i am not robot public key
2. google recaptcha version 2 i am not robot private key
3. the page slug (the page we want to protect using recaptcha)

Now when you browse the page (the added slug page) it will show the captcha and after solve the captcha you will see the orginal page contents.
